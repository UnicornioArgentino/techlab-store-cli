# TechLab Store CLI — DS Análisis

CLI en **Node.js 18+** para gestionar productos de la **FakeStore API**. Diseñado con **arquitectura modular** y enfoque de **portfolio**. Incluye extras para **filtrar por categoría** y **exportar a CSV**.

---

## Requisitos
- Node.js **18+** (incluye `fetch` nativo)
- Windows PowerShell / CMD / bash

---

## Instalación
```powershell
# Parate en la carpeta raíz del proyecto
cd D:\techlab-store-cli\techlab-store-cli
npm install
